React é uma biblioteca JavaScript de código aberto para construir interfaces de
usuário (IU). Ela é uma biblioteca declarativa, o que significa que você
descreve como sua IU deve ser, e o React se encarrega de renderizá-la.

React é uma biblioteca muito popular, usada por uma grande variedade de
empresas, incluindo Facebook, Instagram, Netflix e Uber. É também uma biblioteca
muito poderosa, capaz de construir interfaces de usuário complexas e
responsivas.

## **O que é React?**

React é uma biblioteca JavaScript que usa uma abordagem declarativa para
construir interfaces de usuário. Isso significa que você descreve como sua IU
deve ser, e o React se encarrega de renderizá-la.

React é baseado em componentes, que são unidades de código reutilizáveis que
podem ser usadas para construir sua IU. Cada componente é responsável por
renderizar um pedaço da IU, como um botão, um campo de texto ou um menu.

React também usa uma abordagem de estado para gerenciar os dados de sua IU. O
estado é usado para armazenar os dados que mudam com o tempo, como a seleção de
um usuário em um menu ou a quantidade de itens em um carrinho de compras.

## **Vantagens do React**

React oferece uma série de vantagens, incluindo:

- **Declarativo:** React é uma biblioteca declarativa, o que significa que você
  descreve como sua IU deve ser, e o React se encarrega de renderizá-la. Isso
  torna o código React mais fácil de entender e manter.
- **Componentes:** React é baseado em componentes, que são unidades de código
  reutilizáveis que podem ser usadas para construir sua IU. Isso torna o código
  React mais modular e fácil de manter.
- **Estado:** React usa uma abordagem de estado para gerenciar os dados de sua
  IU. Isso torna o código React mais reativo e fácil de manter.
- **Performance:** React é uma biblioteca muito eficiente, capaz de renderizar
  interfaces de usuário complexas e responsivas.

## **Como usar React**

Para começar a usar React, você precisará instalar a biblioteca no seu projeto.
Você pode fazer isso usando o seguinte comando:

```
npm install react
```

Depois de instalar a biblioteca, você pode criar um novo componente React usando
a seguinte sintaxe:

```tsx
const MyComponent = () => {
  return (
    <div>
      Olá, mundo!
    </div>
  );
};
```

Este componente renderiza uma simples página web com o texto "Olá, mundo!".

Você pode então usar o componente MyComponent em sua aplicação web. Para fazer
isso, você pode importar o componente no seu arquivo principal e usar a tag
`<MyComponent />` para renderizá-lo.

## **Exemplo de código**

O código a seguir mostra um exemplo completo de uma aplicação web React:

```tsx
import React, { useState } from "react";

const App = () => {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>Olá, mundo!</h1>
      <button onClick={() => setCount(count + 1)}>
        +1
      </button>
      <p>O número atual é: {count}</p>
    </div>
  );
};

export default App;
```

Este aplicativo renderiza uma simples página web com um botão que incrementa um
contador. O estado do contador é gerenciado usando a função `useState()`.
