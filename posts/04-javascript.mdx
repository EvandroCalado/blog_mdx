## **História:**

JavaScript foi criado por Brendan Eich em 1995, enquanto ele trabalhava na
Netscape Communications Corporation. Eich originalmente chamou a linguagem de
"Mocha", mas foi renomeada para "JavaScript" em 1995, para capitalizar na
popularidade do Java.

## **Características:**

JavaScript é uma linguagem de programação dinâmica, o que significa que os tipos
de variáveis não são declarados explicitamente. Isso torna JavaScript uma
linguagem fácil de aprender para iniciantes.

JavaScript é também uma linguagem de programação orientada a objetos, o que
significa que os programas JavaScript podem ser organizados em objetos. Isso
torna JavaScript uma linguagem poderosa e flexível.

## **Vantagens:**

JavaScript oferece uma série de vantagens, incluindo:

- **É uma linguagem de script:** Isso significa que pode ser facilmente
  incorporado em páginas da web.
- **É uma linguagem de propósito geral:** Pode ser usado para uma variedade de
  tarefas, incluindo a criação de formulários, animações, jogos e muito mais.
- **É uma linguagem fácil de aprender:** É uma linguagem de programação dinâmica
  e orientada a objetos, o que a torna fácil de aprender para iniciantes.

## **Exemplo de código:**

O código a seguir mostra um exemplo simples de um programa JavaScript:

```tsx
// Este programa imprime "Olá, mundo!" na tela.

function helloWorld() {
  console.log("Olá, mundo!");
}

helloWorld();
```

## **Conclusão:**

JavaScript é uma linguagem de programação poderosa e flexível que pode ser usada
para adicionar interatividade e funcionalidade às páginas da web. É uma
linguagem fácil de aprender e usar, o que a torna uma boa escolha para
iniciantes.
